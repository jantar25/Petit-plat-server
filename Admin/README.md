## Tech Stack

**Frontend:** HTML5 / CSS3 / ReactJS

## Quick start

Clone the repo

```bash
  git clone https://github.com/jantar25/Admin_dashboard
```

Install React-Dashboard with npm

```bash
  cd Admin_dashboard
  npm install
```

## Run Locally

To run locally, run the following command

```bash
  npm start
```

## Deployment

To create a production build

```bash
  npm run build
```
